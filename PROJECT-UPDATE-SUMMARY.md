**Title:** Hockey Rink  
**Type:** Software as a Service (SaaS) Solution  
**Status:** Development in Progress (2025)

**Description:**
A comprehensive SaaS solution for hockey rink small business management that streamlines operations including registration, payments, league management, and administrative functions.

**Technology Stack:**
- **Backend:** .NET Core 9 API, Entity Framework Core, Azure SQL
- **Frontend:** Angular 20, Bootstrap 5.3.3
- **DevOps:** GitHub Actions (CI/CD), Azure Static Web Apps

**Key Features:**
1. User registration with credit card payment processing
2. Skill-level based league session selection
3. Hockey team creation and league management
4. Comprehensive administrative dashboard
